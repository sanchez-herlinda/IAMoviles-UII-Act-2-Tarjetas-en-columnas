# myapp


## Prompt Negocio

El sistema debe generar un archivo main.dart en Flutter con un widget sin estado llamado PaginaRecomendaciones, utilizando el paquete google_fonts para aplicar la tipografía Playfair Display y definiendo estáticamente tres colores: fondo blanco hueso (0xFFF9F9F9), detalle rojo oscuro (0xFF8B0000) y texto gris muy oscuro (0xFF1A1A1A). La estructura principal debe ser un Scaffold que asigne el color de fondo y posea un AppBar rojo oscuro sin elevación, con un botón de retroceso blanco a la izquierda y un título centrado compuesto por un Row alineado al inicio que contenga los textos "La" y "ROMA" en distintos tamaños. El cuerpo de la pantalla debe utilizar un SingleChildScrollView que albergue un Padding de 24.0, dentro del cual irá una Column alineada a la izquierda (CrossAxisAlignment.start) que contenga el título "Recomendaciones", seguido de un Stack que muestre una imagen de red con bordes redondeados (incluyendo un errorBuilder para fallos de carga) y dos botones de íconos de flechas (izquierda y derecha) superpuestos en los laterales centrados verticalmente usando Positioned.fill y Align. Debajo del Stack, la columna debe integrar un widget Center con un ícono de estrella rojo oscuro, el nombre del platillo, una descripción de texto con 80% de opacidad usando withAlpha(204), y un ElevatedButton de 55 de altura y ancho infinito con fondo gris oscuro, bordes redondeados de 15 y el texto "Añadir a carrito". Finalmente, el diseño debe incorporar un pie de página estático en la propiedad bottomNavigationBar mediante un SafeArea y un Container del color del fondo, el cual contendrá una Column de tamaño mínimo conformada por un Divider superior de 1 pixel de grosor en color rojo oscuro y un widget Center con padding vertical que envuelva un texto vacío estilizado.
